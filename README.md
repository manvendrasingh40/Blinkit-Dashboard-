An interactive Power BI dashboard designed to analyze Blinkit's sales performance, customer ratings, outlet distribution, and product trends. The dashboard provides key business insights through dynamic visualizations and slicers, enabling users to explore data across different outlet sizes, locations, and item categories.

Item Type

🎯 Project Objective

The goal of this dashboard is to provide stakeholders with a centralized view of Blinkit's operational and sales performance, helping identify growth opportunities, optimize inventory decisions, and improve outlet-level strategies.
